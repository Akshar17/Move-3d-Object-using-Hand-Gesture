Steps to Execute

First run python file 'handapi.py'

Step 1: Press Start Recording
Step 2: Double click on you palm
Step 3: Move your palm
Step 4: Press 'q' key to terminate frame
Step 5: You will be able to see object moving
Step 6: Press Reset button
Step 7: Repeat Steps to load again

Terminate 'handapi.py' by ctrl+c or cmd+c
